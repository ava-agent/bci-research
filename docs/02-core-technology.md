# BCI 核心技术栈

## 1. 系统架构概览

一个完整的 BCI 系统由以下核心模块组成：

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  信号采集    │ → │  信号预处理   │ → │  特征提取    │ → │  分类/解码   │ → │  应用输出    │
│  Acquisition │    │ Preprocessing │    │  Extraction  │    │ Classification│    │  Application │
└─────────────┘    └──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
     ↑                                                                              │
     └──────────────────────── 反馈（Feedback Loop）────────────────────────────────┘
```

## 2. 信号采集技术

### 2.1 电极技术

**干电极 vs 湿电极：**

| 类型 | 导电介质 | 阻抗 | 舒适度 | 信号质量 | 使用场景 |
|------|---------|------|--------|---------|---------|
| 湿电极 | 导电凝胶 | 低（<5kΩ） | 较差 | 高 | 研究/临床 |
| 干电极 | 直接接触 | 高（>100kΩ） | 好 | 中 | 消费级/日常 |
| 半干电极 | 少量液体 | 中 | 较好 | 较高 | 折中方案 |

**植入式电极：**
- **微电极阵列（Utah Array）**：96 通道硅基阵列，BrainGate 使用的核心硬件
- **柔性电极**：Neuralink 的超细柔性线程（直径 ~5μm），减少组织损伤
- **血管内支架电极**：Synchron Stentrode，通过颈静脉植入，无需开颅

**前沿电极技术（2025）：**
- Columbia 大学微型芯片：仅 3mm³ 体积，65,536 个电极，1,024 记录通道，16,384 刺激通道
- 磁微测量（Magnetomicrometry）：在肌肉组织中植入微型磁铁，通过磁场传感器追踪

## 3. 信号预处理

### 3.1 去噪与滤波

原始脑电信号包含多种噪声源，需要逐步处理：

```
原始信号 → 陷波滤波(50/60Hz工频) → 带通滤波(0.5-100Hz) → 伪迹去除 → 重参考 → 分段
```

**常用滤波技术：**
- **带通滤波（Band-pass Filter）**：保留感兴趣的频率范围
- **陷波滤波（Notch Filter）**：去除电源线干扰（50Hz/60Hz）
- **自适应滤波（Adaptive Filter）**：实时跟踪和去除变化的噪声

### 3.2 伪迹去除

| 伪迹类型 | 来源 | 去除方法 |
|---------|------|---------|
| 眼动伪迹（EOG） | 眨眼、眼球运动 | ICA、回归法 |
| 肌电伪迹（EMG） | 面部/颈部肌肉 | ICA、CCA、小波变换 |
| 心电伪迹（ECG） | 心跳 | ICA、自适应滤波 |
| 运动伪迹 | 头部运动 | 加速度计辅助、ASR |

**核心方法：**
- **独立成分分析（ICA）**：将混合信号分离为独立成分，识别并移除伪迹成分
- **小波变换去噪（Wavelet Denoising）**：在时频域中去除噪声
- **伪迹子空间重构（ASR）**：用于实时在线伪迹去除

## 4. 特征提取

### 4.1 时域特征
- 均值、方差、偏度、峰度
- Hjorth 参数（Activity, Mobility, Complexity）
- 自回归模型系数（AR Coefficients）

### 4.2 频域特征
- 功率谱密度（PSD）
- 各频带能量比（Delta: 0.5-4Hz, Theta: 4-8Hz, Alpha: 8-13Hz, Beta: 13-30Hz, Gamma: 30-100Hz）
- 谱熵

### 4.3 时频域特征
- 短时傅里叶变换（STFT）
- 小波变换（Wavelet Transform）
- 希尔伯特-黄变换（HHT）

### 4.4 空间特征
- **共空间模式（CSP, Common Spatial Patterns）**：运动想象 BCI 中最经典的空间滤波方法
- 拉普拉斯参考
- 波束成形（Beamforming）

### 4.5 连接性特征
- 相干性（Coherence）
- 相位锁定值（PLV）
- 格兰杰因果（Granger Causality）

## 5. 分类与解码算法

### 5.1 传统机器学习方法

| 算法 | 优点 | 缺点 | 典型应用 |
|------|------|------|---------|
| **LDA** | 计算简单、实时性好 | 假设数据线性可分 | 在线 BCI |
| **SVM** | 小样本表现好 | 核函数选择难 | MI 分类 |
| **随机森林** | 抗过拟合 | 特征数多时慢 | 多类分类 |
| **KNN** | 简单直观 | 维度灾难 | 基础分类 |

### 5.2 黎曼几何方法（Riemannian Geometry）

近年来在 BCI 领域表现出色的方法：
- 将 EEG 协方差矩阵映射到黎曼流形
- 在切空间（Tangent Space）中进行分类
- 天然适合处理 EEG 的对称正定矩阵特性
- 代表工具：pyRiemann 库

### 5.3 深度学习方法

| 模型架构 | 特点 | BCI 应用 |
|---------|------|---------|
| **CNN** | 自动学习空间-时间特征 | EEGNet, ShallowConvNet, DeepConvNet |
| **LSTM/GRU** | 捕捉时序依赖 | 序列解码、语音 BCI |
| **Transformer** | 自注意力机制 | EEG-Conformer, BCI-Transformer |
| **GAN** | 数据增强 | 生成合成 EEG 数据 |
| **VAE** | 潜在空间表征 | 特征学习 |
| **图神经网络** | 建模电极连接关系 | 基于脑网络的分类 |

**代表性深度学习模型：**
- **EEGNet**：轻量级 CNN，使用深度可分离卷积，参数少且泛化好
- **Braindecode**：基于 PyTorch 的深度学习 BCI 工具包
- **多模态 BCI 解码**：结合 AI 的多模态方法（2025 前沿方向）

### 5.4 迁移学习（Transfer Learning）

解决 BCI 中跨被试（Cross-Subject）和跨会话（Cross-Session）问题的关键技术：
- 域自适应（Domain Adaptation）
- 预训练+微调
- 元学习（Meta-Learning）

## 6. 当前技术挑战

| 挑战 | 描述 | 当前进展 |
|------|------|---------|
| 低信噪比 | 非侵入式信号穿越头皮/颅骨后衰减严重 | 改进电极和算法 |
| 非平稳性 | 脑电信号随时间、状态变化 | 自适应算法、迁移学习 |
| 个体差异 | 不同人脑结构和信号模式差异大 | 校准优化、跨被试迁移 |
| 实时性 | 在线系统需要低延迟处理 | 轻量模型、边缘计算 |
| 长期稳定性 | 植入电极的生物相容性退化 | 新材料、柔性电极 |
| 多类解码 | 从有限信号中解码丰富的意图 | 多模态融合 |
