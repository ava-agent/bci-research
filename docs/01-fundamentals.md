# 脑机接口基础概念与技术分类

## 1. 什么是脑机接口（BCI）

脑机接口（Brain-Computer Interface, BCI）是一种测量大脑活动并将其实时转化为功能性输出的系统，改变大脑与外部或内部环境之间的交互方式。BCI 通过采集、处理和解码神经信号，使用户能够仅凭意念控制外部设备（如计算机、机械臂、轮椅等），无需依赖肌肉运动。

### 核心工作流程

```
大脑活动 → 信号采集 → 信号预处理 → 特征提取 → 分类/解码 → 设备控制/反馈
```

## 2. 发展历程

| 年代 | 里程碑 |
|------|--------|
| 1924 | Hans Berger 首次记录人类脑电图（EEG） |
| 1973 | Jacques Vidal 首次提出 "Brain-Computer Interface" 概念 |
| 1998 | 首例人体侵入式 BCI 植入（Philip Kennedy） |
| 2004 | BrainGate 首次人体临床试验 |
| 2006 | 瘫痪患者首次通过 BCI 控制计算机光标 |
| 2016 | 非侵入式 BCI 实现脑控无人机 |
| 2020 | Neuralink 展示猪脑植入芯片 |
| 2023 | Synchron 获 FDA 突破性设备认定 |
| 2024 | Neuralink 首例人体植入手术成功 |
| 2025 | 语音 BCI 精度达 99%，约 25 项临床试验进行中 |
| 2026 | BCI 进入规模化生产和商业化前夜 |

## 3. 技术分类

脑机接口根据**信号采集电极的放置位置**分为三大类：

### 3.1 侵入式（Invasive）

将电极直接植入大脑皮层或脑组织内部。

**信号类型：**
- 皮层脑电图（ECoG, Electrocorticography）
- 单神经元动作电位（Spike / Single-Unit Activity）
- 局部场电位（LFP, Local Field Potential）

**特点：**
- 最高的时空分辨率和信号精度
- 可记录单个神经元级别的活动
- 需要开颅手术，存在感染和组织损伤风险
- 长期稳定性面临挑战（电极退化、瘢痕组织）

**代表产品：** Neuralink N1 芯片、BrainGate 微电极阵列、Paradromics Connexus

### 3.2 半侵入式 / 介入式（Semi-Invasive）

电极植入在硬脑膜上/下或血管内，不直接穿透脑实质。

**信号类型：**
- 硬膜外皮层脑电图（Epidural ECoG）
- 血管内脑电图（Endovascular EEG）

**特点：**
- 平衡了信号质量和手术创伤
- 空间分辨率优于非侵入式，低于侵入式
- 免疫反应和瘢痕组织产生的概率较小
- 可利用现有血管介入手术技术

**代表产品：** Synchron Stentrode（血管内支架电极）、Precision Neuroscience 超薄皮层带

### 3.3 非侵入式（Non-Invasive）

在头皮表面或附近采集大脑信号，无需手术。

**信号类型：**

| 技术 | 全称 | 原理 | 时间分辨率 | 空间分辨率 |
|------|------|------|-----------|-----------|
| **EEG** | 脑电图 | 头皮电极记录电位变化 | ~毫秒级 | >5mm |
| **fNIRS** | 功能近红外光谱 | 近红外光测量血氧变化 | ~秒级 | ~1cm |
| **MEG** | 脑磁图 | 超导量子干涉仪测磁场 | ~毫秒级 | ~2-3mm |
| **fMRI** | 功能核磁共振 | BOLD 信号测量血氧 | ~秒级 | ~1mm |

**特点：**
- 低成本、低风险、操作简单
- 用户接受度远高于侵入式
- 信号需穿越头皮和颅骨，信噪比低（EEG 约 -15~5dB）
- 空间分辨率受限
- 最适合大规模消费级应用

**代表产品：** OpenBCI 系列、Emotiv EPOC/Insight、Muse 2/Muse S、Neurosity Crown

## 4. 三类 BCI 技术对比

| 对比维度 | 侵入式 | 半侵入式 | 非侵入式 |
|----------|--------|---------|----------|
| 信号质量 | 极高 | 高 | 中低 |
| 空间分辨率 | 单神经元级 | 毫米级 | >5mm |
| 手术风险 | 高（开颅） | 中（介入） | 无 |
| 长期稳定性 | 挑战大 | 较好 | 好 |
| 成本 | 极高 | 高 | 低 |
| 商业化阶段 | 临床试验期 | 临床试验期 | 已商业化 |
| 用户接受度 | 低 | 中 | 高 |
| 典型应用 | 瘫痪康复、语音恢复 | 运动功能恢复 | 神经反馈、游戏、教育 |

## 5. BCI 范式（Paradigms）

BCI 系统常用的信号范式包括：

- **运动想象（Motor Imagery, MI）**：用户想象肢体运动，产生特定频带的 EEG 变化
- **P300 事件相关电位**：对目标刺激产生的约 300ms 延迟正波，用于拼写器等
- **稳态视觉诱发电位（SSVEP）**：注视不同频率闪烁的视觉刺激，诱发对应频率的脑电响应
- **慢皮层电位（SCP）**：通过训练调节脑电的缓慢电位变化
- **语音想象（Speech Imagery）**：想象说话时的神经活动模式，用于语音解码
